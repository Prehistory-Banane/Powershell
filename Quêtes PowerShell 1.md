# Équivalent des commandes Shell Bash et PowerShell  

* cp > Copy-Item  
* rm > Remove-Item  
* cd > Set-Location  
* mkdir > New-item -ItemType Directory  
* man > Get-Help  
* history > Get-History  
* alias > Get-Alias  
* cat > Get-Content


# Les commandes PowerShell  utilisées pour faire mes recherches  
`get-help cp`  
`get-help rm`  
`get-help cd`  
`get-help mkdir`  
`get-help man`  
`get-help history`  
`get-help alias`  
`get-help cat`
